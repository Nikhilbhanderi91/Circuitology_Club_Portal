# Circuitology Club Portal

Welcome to the official web portal for the **Circuitology Club** under the Department of Information Communication Technology at Marwadi University (MU). 

This portal has been engineered from the ground up with a premium **Cybernetic & Electronics Hardware aesthetic**, featuring modern glassmorphism, responsive grids, glowing neon elements, and high-performance interactive animations.

---

## 🚀 Key Features

*   **Dynamic Circuit Canvas**: The landing page hero section features an interactive HTML5 `<canvas>` background simulating real-time electrical signal pulses traversing a randomized circuit board node grid.
*   **Next-Gen Cybernetic Design System**: Built using curated color palettes (deep space voids, electric neon cyan, cyber purple, and glowing amber details) with typography driven by **Orbitron** (display elements) and **Inter** (body copy).
*   **Projects Showcase**: A comprehensive collection of hardware innovation lab projects (including Arduino, ESP32, ATmega32, and solar-power systems) with built-in schematic inspection modal overlays.
*   **Activity & Training Scheduler**: Modern timeline cards displaying upcoming workshops (PCB auto-routing, IoT, microchip programming) alongside active winner leaderboards for finished events.
*   **Unified Member Access**: An elegantly animated sliding login and registration interface to manage user access and workshop registrations.

---

## 🛠️ Technology Stack

*   **Front-End**: Semantic HTML5, Vanilla CSS3 (Custom Design Tokens, Flexbox, Grid Layouts), and Vanilla JavaScript (Canvas API, DOM manipulation).
*   **Back-End**: PHP (Access authentication management and backend logic).
*   **Styling Theme**: Cyberpunk / Cybernetic Glassmorphic UI.

## ⚙️ Local Setup & Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Nikhilbhanderi91/Circuitology_Club_Portal.git
    cd Circuitology_Club_Portal
    ```

2.  **Run via Local PHP Server**:
    Ensure you have PHP installed, then boot up a local testing server in the root directory:
    ```bash
    php -S localhost:8000
    ```

3.  **Access the Portal**:
    Open your web browser and navigate to:
    ```text
    http://localhost:8000
    ```
